### How to Get GitHub Raw Links

The general format for GitHub raw links is:

```
https://raw.githubusercontent.com/{owner}/{repo}/{branch}/{path-to-file}
```

Where:
- `{owner}` = `adamalama151212-code`
- `{repo}` = `songs`
- `{branch}` = `main` (or the branch where the file is located)
- `{path-to-file}` = the path to the file from the repository root
